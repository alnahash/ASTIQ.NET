# astiq.net

Marketing site and legal pages for **ASTIQ Cryptogram**, a word-puzzle game.

Static site deployed on Vercel.

| Path | File | Purpose |
|---|---|---|
| `/` | `index.html` | Landing page |
| `/privacy` | `privacy.html` | Privacy policy (Google Play requirement) |
| `/app-ads.txt` | `app-ads.txt` | AdMob authorized sellers (fill in after AdMob account exists) |

`vercel.json` enables clean URLs (`/privacy` → `privacy.html`).
